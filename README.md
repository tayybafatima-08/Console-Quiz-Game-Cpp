# Console Based Quiz Game (C++)

## Overview
This project is a Console Based Quiz Game developed in C++ as a Programming Fundamentals final project.

The application allows users to attempt multiple choice questions, validates input, and dynamically calculates scores.

## Features
- Interactive console interface
- Multiple choice question system
- Score tracking
- Input validation
- Modular implementation using functions

## Concepts Applied
- Control structures (if-else, loops)
- Functions
- Arrays
- Logical problem solving
- File handling
